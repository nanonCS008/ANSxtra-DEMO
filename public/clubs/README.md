# Club Images

Place club images here with filenames matching the club ID in `data/clubs.json`.

## Required Images

- `operation-smile.jpg`
- `school-production.jpg`
- `mun.jpg`
- `interact.jpg`
- `eco.jpg`
- `dofe.jpg`
- `unicef.jpg`
- `tedx.jpg`

## Image Guidelines

- **Format**: JPG or PNG (JPG preferred for smaller file size)
- **Dimensions**: Recommended 800x600px or 4:3 aspect ratio
- **Size**: Keep under 200KB for optimal loading
- **Content**: Should represent the club's activities or logo

If an image is missing, a gradient placeholder will be shown automatically.
